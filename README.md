## 프로젝트 소개
```
간단한 연습용 SNS사이트 제작
```

## 주요기능
```
- 게시글 작성
- 좋아요 
- 리트윗 
- 댓글(쓰기, 조회)
- 팔로우/언팔로우 
- 관리자 기능(게시글 삭제, 신고처리, 계정삭제)
- 해시태그
```

## 실행 방법
```bash
pip install -r requirements.txt
streamlit run app.py
```
## 사용 방법
```
1. 회원가입 or 로그인
2. 게시글 확인
3. 게시글 작성 
```

## 기술 스택
```
streamlit, pandas
```
